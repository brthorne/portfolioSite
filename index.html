About

My professional interests include player skill modeling and training, story and environment generation, making robots make mistakes, and doing science on whiteboards.

FireBolt

FireBolt is a fully scriptable cinematic sequencer using Unity 5. It controls animation and movement of actors as well as positioning, orientation, and movement of virtual cameras. FireBolt has been used for cinematic realization in a planning-based narrative generator as well as a more conventional cinematic sequencer.

Donjon Skinner

Donjon Skinner is tile-based map texturing interface developed in Unity5. It supports a subset of the map features given by the donjon 5e generator and allows users to import arbitrary textures, assign them to tile types, and export textured maps.

ProcDefense

ProcDefense is an arcade-style game created as a research platform for player skill training. It provides an interface for dynamically adjusting game difficulty along any properties of the particle system. Current work seeks to establish a progression of values that constitute monotonically increasing difficulty via user study. Future work will explore player and pedagogical models to inform difficulty adjustment.
